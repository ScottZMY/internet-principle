Domain name system

Web server’s IP address are associated with their domain name. When entering in a domain name, the request will go to a DNS server usually associated with the ISP. The DNS server will check if it has the address saved. If not, it will ask other DNS server’s and then return the address

Cookie

Cookies are pieces of data downloaded your computer when visiting a website. Some cookies track if you have logged into a secure website. while others can track if you were online.And some cookies store sensitive data such as credit card number. Cookies can be declined by user or removed from the computer. These action can be down with your browser.
